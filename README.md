Superstore Sales Analysis

## Objectif
Analyser les données de ventes du Superstore afin d’identifier :
- les catégories les plus performantes
- les tendances de ventes
- des insights utiles pour la prise de décision

## Outils utilisés
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
